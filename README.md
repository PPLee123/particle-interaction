# Particle Interaction Visual Effect

這是一個基於 HTML5 Canvas 的科技感粒子視覺效果。包含圓形填充、隨機有機流動以及動態的滑鼠互動反應。

![Interaction Recording](https://media.giphy.com/media/your-link-here.gif) <!-- 提示：您可以將錄影上傳後替換此連結 -->

## 功能特點
- **圓形限制**：粒子初始並持續填滿中央圓形區域。
- **有機流動**：在無互動時呈現平滑、隨機的動態效果。
- **滑鼠交互**：
  - 粒子會根據滑鼠移動方向迅速逃離。
  - 感應滑鼠速度，動態增加逃逸力。
- **自動回歸**：滑鼠移開後，粒子會自動平滑地回到原始圓形範圍。

## 技術實作與效果說明

### 1. 自然流動 (Organic Flow)
粒子在沒有外部干預時，使用 `Math.sin()` 與 `Math.cos()` 結合隨機的 `noiseOffset` 來產生輕微的位移力。
- **邏輯**：隨時間變化的正弦函數模擬出流體般的漂移感，而非單純的隨機震動。
- **效果**：粒子會像在液體中一樣緩慢且有機地遊走。

### 2. 滑鼠排斥交互 (Mouse Repulsion)
當移動滑鼠時，系統會計算粒子與光標的距離。
- **逃離力**：距離越近，受到的排斥力愈大。
- **動態速度**：我們紀錄了滑鼠的移動速度 (`mouse.vx`, `mouse.vy`)，將其部分動量傳遞給粒子。
- **效果**：當您快速划過粒子群時，粒子會根據您划動的方向噴發，呈現強烈的實體互動感。

### 3. 自動回歸 (Spring-back Logic)
每個粒子都記得自己的初始位置 (`baseX`, `baseY`)。
- **恢復力**：粒子與初始位置的距離越大，受到的向心力越強，類似於彈簧效果。
- **摩擦阻尼**：速度會乘以 `0.92` 的摩擦係數，確保粒子最終會靜止下來而不會永久晃動。

### 4. 視覺餘影 (Trailing Effect)
每幀繪製前，我們使用帶有透明度的深色矩形 `rgba(10, 10, 12, 0.2)` 覆蓋全螢幕，而不是直接清空畫布。
- **效果**：粒子移動時會留下淡淡的尾跡，增加視覺的流暢感。
